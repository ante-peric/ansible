Ansible System Modules:

Packaging service Module - Documentation

Informacije o service modulu:

$ ansible-doc service
Parameter Choices/Defaults Comments
name
string/required		 Ime Servisa
enabled
boolean		 Choices:

no
yes		 Osigurava da li će service startati nakon boot procesa
Najmanje jedno stanje je potrebno.
state
string		 Choices:

reloaded
restarted
started
stopped		 started/stopped su "idempotent" radnje koje neće izvršavati naredbe osim ako je potrebno
restarted - Restartirat će servis
reload - Uvijek će reload
Najmanje jedno stanje je potrebno
Napomena: reload će startati servis ako već nije startan

Primjeri:

- name: Start service httpd, if not started
service:
name: httpd
state: started


- name: Stop service httpd, if started
service:
name: httpd
state: stopped


- name: Restart service httpd, in all cases
service:
name: httpd state: restarted


- name: Reload service httpd, in all cases
service:
name: httpd
state: reloaded


- name: Enable service httpd, and not touch the state
service:
name: httpd
enabled: yes


- name: Start service foo, based on running process /usr/bin/foo
service:
name: foo pattern: /usr/bin/foo
state: started


- name: Restart network service for interface eth0
service:
name: network
state: restarted
args: eth0
